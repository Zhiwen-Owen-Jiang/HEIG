# Highly-Efficient Imaging Genetics (HEIG)
`HEIG` is a statistical framework for conducting joint analysis for large-scale imaging and genetic data. It utilized spatial correlation within images to enhance both computational and statistical efficiency. Currently, `HEIG` is capable of performing genome-wide association analysis (GWAS) on images for unrelated subjects, as well as heritability analysis for each grid point, genetic correlation analysis for a pair of grid points, and cross-trait genetic correlation between a grid point and an external phenotype (such as complex disorders) using summary statistics.
